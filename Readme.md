### Build on macOS

**Requirements**

- v8 embed static library `libv8_monilith.a`
- glfw3 (install using `brew install glfw3`)
- clang and llvm

**Flags**
- BK_ENABLE_SHADER_TRANSLATOR: _default to ON_ Use tranlator from Angle Project to transpile shader
